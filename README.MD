# Aplikacja do przeliczania paliwa.
Rozbudowana aplikacja która posiada 3 główne opcje
  - Kalkulator potrzebnego paliwa
  - Kalkulator średniego spalania 
  - Kalkulator dystansu 

## Główny cel narzędzia
Głównym celem aplikacji było proste liczenie wszystkiego co potrzebne dla kierowców w kwestii przeliczania paliwa.

## Zakres funkcjonalności 
  #### W aplikacji możemy
  - Ustalić jednostkę paliwa [Litry, galon angielski, galon amerykański]
  - Ustalić jednostkę odległości [Kilometry, mile]
  - Obliczyć potrzebne paliwo 
  - Obliczyć średnie spalanie 
  - Obliczyć dystans 
  - Cenę paliwa lub cenę za przejazd 

  #### podczas podawanie informacji możemy zobaczyć czy dane są prawidłowe, w polu (uzupełnianie wartości) zmieni się ikona.
  - Dla złej wartości - czerwona
  - Dla dobrej wartości - zielona

## Technologie 
  - HTML5
  - CSS3
  - SCSS
  - JavaScript
